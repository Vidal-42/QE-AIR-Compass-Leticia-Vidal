## Repositório

Este repositório foi criado com o objetivo de documentar, organizar e versionar as atividades desenvolvidas durante o programa **AIR (Academic Internship Program)** da **CompassUOL**, na trilha de **Quality Engineering & AI**.

Aqui estão reunidas as evidências, exercícios e artefatos produzidos ao longo das **6 sprints da Fase 1 (Estágio)**, posteriormente estruturados durante a **Fase 2 (Trainee)** como parte do processo de organização, padronização e versionamento utilizando Git e GitHub.

Este ambiente serve como um repositório central de documentação das práticas, atividades e conhecimentos aplicados ao longo do programa.

---

## Objetivos

- Documentar as atividades realizadas durante a Fase 1 do estágio  
- Centralizar planos de teste, evidências, relatórios e exercícios  
- Aplicar boas práticas de versionamento com Git e GitHub  
- Manter um histórico estruturado das atividades desenvolvidas  
- Organizar os artefatos de Quality Engineering de forma clara e rastreável
